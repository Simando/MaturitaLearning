# Maturita Learning
Program na učení odpovědí k otázkám ze sítí a programování k maturitě

## Stažení a spuštění
  1. Stáhni složku MaturitaLearning
  2. Extrahuj jí ze .zip formátu
  3. Otevři node.js
  4. Přejdi do umístění složky
  5. Zadej příkaz "npm start"
  6. Hodně štěstí!
